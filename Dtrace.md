
### 常用命令：

```sh
$ sudo dtrace -n 'proc:::signal-send /pid/ { printf("%s -%d %d",execname,args[2],args[1]->pr_pid); ustack(); }'

$ sudo dtrace -n 'proc:::exec-success { printf("%Y %s\n",walltimestamp,curpsinfo->pr_psargs); ustack(); }'
```
